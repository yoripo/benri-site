# べんりツール

シンプルな日本語ユーティリティ集サイトです。

## ツール一覧

### 日付計算 (`index.html`)
- 今日からN日後（マイナスで前も計算可）
- 2つの日付の間の日数

### 和暦西暦 (`wareki.html`)
- 和暦 ↔ 西暦 リアルタイム変換
- 明治〜令和の全年一致確認表（元号フィルタ・西暦検索つき）

## 技術スタック

- HTML / vanilla JS / CSS（モック段階）
- 移行予定: Next.js + TypeScript + Tailwind CSS
- ホスティング予定: Vercel

## ローカルで開く

```bash
git clone https://github.com/yoripo/benri-site.git
cd benri-site
# index.html または wareki.html をブラウザで開く
```
